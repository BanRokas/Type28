Duomenų tipai

  • Number - skaičius - (nebūtina: int(integer) - sveikasis, float - realusis)
  • String - tekstas - (nebūtina: char(character) - vienas simbolis) (char[])

  • Array - masyvas - konteineris (eiliškumas svarbus)
  • Object - objektas - konteineris (eiliškumas nesvarbus)
  • NaN, null, undefined, Symbol...

  • Inline - JS event'o aprašymas elemento atributo skiltyje
  • Internal - HTML kode atidaryti script tag'ą ir jo viduje rašyti JS kodą
  • External - Sukurti atskirą .js failą ir jį prikabinti prie HTML'o su script tag'u

Kintamieji

Kintamasis - kažkoks sugalvotas vardas ir jam priskirta informacija. Kreipiantis į tą vardą - galime išgauti jo viduje slypinčią informaciją.

Kintamojo kūrimas

  1. kintamojo startas
    • var - senasis būdas (nebenaudotinas) (hoist'inimo pliusas)
    • let - naujasis būdas (kintamas)
    • const - naujasis būdas (nekintamas)
    • - nieko nerašant (NENAUDOTI)
  2. kintamojo vardas - susigalvoji logišką ir pakankamai trumpą vardą be tarpų ir/ar lietuviškų simbolių
  3. = - priskyrimo simbolis
  4. reikšmė
    • number - 4 | 0 | -5 | 1.51 | -3.3 | 12-4...
    • string - 'Žodis' | 'keli žodžiai' | "" | ``...
    • boolean - true | false
    • array - ['zodis', 54, false, [], {}, 4, ...]
    • object - { raktinisZodis: reiksme, kitasRakZodis: kitaReiksme, ... }
    • ...

Aritmetika

  • x+y - sudėtis
  • x-y - atimtis
  • x*y - daugyba
  • x/y - dalyba
  • x%y - liekana
  • laipsnio kėlimas
    • x**y
    • Math.pow(x, y)
  • šaknies traukimas
    • x**(1/y)
    • Math.sqrt(x) - kvadratinė šaknis
    • Math.cbrt(x) - kūbinė šaknis
    • Math.pow(x, 1/y)

Palyginimai

  • x > y - ar x daugiau negu y
  • x < y - ar x mažiau negu y
  • x >= y - ar x daugiau arba lygu negu y
  • x <= y - ar x mažiau arba lygu negu y
  • x == y - ar x lygus y (reikšmė)
  • x === y - ar x lygus y (reikšmė ir tipas)
  • x != y - ar x nelygus y (reikšmė)
  • x !== y - ar x nelygus y (reikšmė ir tipas)

Loginiai operatoriai

  • ! - NOT - apverčia reikšmę
  • || - OR - bent vienoje iš OR'o pusių turi būti tiesa, kad bendrai grąžintų tiesą
  • && - AND - ar abejose AND'o pusėse yra tiesa - tik tuomet grąžins tiesą

Tipo operatoriai

  • typeof(kazkas) - tikrina koks yra "kazkas" tipas
  • object instanceof Class - ar objektas buvo sukurtas naudojant nurodytą Klasę
  • "keyword" in Object - ar objektas turi nurodytą raktinį žodį

Sąlygos

Jeigų sąlyga yra tiesa - jos viduje parašytas kodas yra vykdomas

  • if'ai
    • if
    • if else
    • if elseif
    • if elseif else
    • if elseif ... elseif else
  • switch - ar tekstas lygus kažkam
  • ternary - sąlyga ? tiesos atvejo kodas : melo atvejo kodas
  • loginės

Ciklai

Tai kažkokia veiksmų seka, kuri vyksta tol, kol prisegta sąlyga grąžina tiesą.

  • for
  • for in - sukti ciklą per objekto raktinius žodžius
  • for of - sukti ciklą per masyvo reikšmes
  • while
  • do while
  • masyvų iteraciniai metodai
